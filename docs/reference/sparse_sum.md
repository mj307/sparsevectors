# Sum of a sparse_numeric Vector

Computes the sum of all nonzero stored values.

## Usage

``` r
sparse_sum(object)

# S4 method for class 'sparse_numeric'
sparse_sum(object)
```

## Arguments

- object:

  A sparse vector.

## Value

The numeric sum.

## Methods (by class)

- `sparse_sum(sparse_numeric)`: Sum stored entries.
