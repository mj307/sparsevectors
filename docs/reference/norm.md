# Norm of a sparse_numeric Vector

Computes the Euclidean (L2) norm using only stored values.

## Usage

``` r
norm(object)

# S4 method for class 'sparse_numeric'
norm(object)
```

## Arguments

- object:

  A sparse vector.

## Value

A numeric norm.

## Methods (by class)

- `norm(sparse_numeric)`: Euclidean norm for sparse vectors.
