# Standardize a sparse_numeric Vector

Converts the sparse vector to dense form, centers it, scales it, and
then converts it back into a sparse representation.

## Usage

``` r
standardize(object)

# S4 method for class 'sparse_numeric'
standardize(object)
```

## Arguments

- object:

  A sparse vector.

## Value

A standardized sparse vector.

## Methods (by class)

- `standardize(sparse_numeric)`: Standardize a sparse vector.
