# Dot Product of Two sparse_numeric Vectors

Computes the sum of pairwise products for overlapping nonzero positions.

## Usage

``` r
sparse_crossprod(x, y)

# S4 method for class 'sparse_numeric,sparse_numeric'
sparse_crossprod(x, y)
```

## Arguments

- x:

  A sparse vector.

- y:

  A sparse vector.

## Value

A numeric scalar.
