# Sort a sparse_numeric Object

Reorders a sparse vector based on the stored positions.

## Usage

``` r
# S4 method for class 'sparse_numeric'
sort(x, decreasing = FALSE, ...)
```

## Arguments

- x:

  A `sparse_numeric` object.

- decreasing:

  Whether to sort in decreasing order of position.

- ...:

  Additional arguments (unused).

## Value

A new sorted `sparse_numeric` vector.
